# Car Lounge Frontend

Next.js App Router проект для Car Lounge.

## Стек

- Next.js App Router
- TypeScript
- Tailwind CSS
- Lucide React
- next/font
- next/image

## Команды

```bash
npm install
npm run dev
npm run build
npm run start
```

## Страницы

- `/` — главная
- `/avtomobili-v-nalichii` — автомобили в наличии
- `/avtomobili-v-nalichii/[slug]` — SEO-страницы авто и брендов
