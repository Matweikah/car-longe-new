# Stock images

В карточках страницы `Автомобили в наличии` убраны локальные SVG/генерированные заглушки. Поле `image` в `data/stock-cars.ts` теперь указывает на реальные фото-источники и внешние изображения автомобилей; для `next/image` добавлены соответствующие remotePatterns.

При следующей доступности полного `yandex.xml` можно заменить эти URL на точные `<images>` из фида без изменения верстки карточек.
